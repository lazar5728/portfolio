## 🚀 Installation et utilisation

1. Clonez le repository:
```bash
git clone https://github.com/lazar5728/portfolio.git
```

2. Ouvrez le fichier `index.html` dans votre navigateur

Ou utilisez un serveur local:
```bash
# Avec Python
python -m http.server 8000

# Avec Node.js (http-server)
npx http-server
```

3. Accédez à `http://localhost:8000`

## 📱 Responsive Design

Le portfolio est entièrement responsive et optimisé pour:
- 📱 Mobile (320px+)
- 📱 Tablette (768px+)
- 💻 Desktop (1024px+)

## 🎨 Personnalisation

Pour personnaliser le portfolio:

1. Modifiez les informations personnelles dans `index.html`
2. Ajustez les couleurs dans `style.css` ou via Tailwind
3. Ajoutez vos projets dans la section projets
4. Remplacez les images dans le dossier `IMG/`
